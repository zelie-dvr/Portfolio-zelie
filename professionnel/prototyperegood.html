Prototype web de la marque Regood

Cliquez ici pour voir le prototype.

J'ai eu l'opportunité de concevoir un prototype web pour la marque Regood. Pour ce projet, j'ai respecté les codes graphiques de la marque tout en intégrant les retours de mes collègues, afin de garantir une expérience utilisateur fluide et engageante. J'ai utilisé Figma pour créer une interface intuitive, en veillant à ce que le design reflète l'identité visuelle de Regood. Le site sera construit sur un template WordPress. Une fois le prototype finalisé, j'ai partagé le lien avec l'équipe pour recueillir des avis supplémentaires. Pour certaines fonctionnalités non disponibles avec le thème WordPress, il sera possible de créer un thème enfant avec du code supplémentaire pour le personnaliser.
